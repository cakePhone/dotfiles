# cake's Aylur's GTK Shell bar

This is a vertical AGS bar themed with catppuccin mocha color scheme.

It uses themed icons from the GTK user set icon theme.

## Building, bundling and restarting the systemd service

AGS is managed by my own systemd service, the build script takes care of bundling and restarting the bar, run it after changes.

```bash
./build.sh
```
